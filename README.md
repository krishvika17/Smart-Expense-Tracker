# Smart Expense Tracker

A Python application that helps users efficiently manage their personal finances by recording, tracking, and analyzing daily expenses using JSON storage. It offers a concise summary of spending patterns of the user to help with smarter budgetary choices.
Key Features:
1) Add expenses with details like amount, category, date, and notes.
2) View all recorded expenses in an organized format.
3) Categorize expenses for easier tracking and analysis
4) It lets the user know where their money was spent the most.
5) Aids in finding expenditure trends and efficiently managing budgets.
